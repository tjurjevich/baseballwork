This project was completed for my master's level deterministic operations research course.

Project objective: Create a Gurobi model to determine the optimal house to hypothetically purchase so as to minimize the number of expected miles driven throughout the week. Various constraints are provided concerning a home's features (listing price, # bedrooms, # bathrooms, and square footage). The objective function contains the number of trips to each "destination" I have listed during a normal week for me. The current code follows the weekly schedule I provided...
